InkForge is a storyteller that makes use of ai in order to generate stories similar to other services such as character.ai and AiDungeon one factor those two services have in common is that the ai behind them and thus all of the user data is run and stored on their servers.
This is where InkForge differs from it's competition, we have no servers to run an ai nor store user data on. Due to advancments in technology particularly in the AI field in the early 20s we have a model that is able to be run entirely within your web browser.
The use of this model means that your stories are entirely yours with no limitations. You can also be sure that your data is secure as the only server involved in the process is the server hosting the website itself.
FOSS (Free Open Source Software) is an important philosophy and ensures that the inner workings of software is freely known as well as ensuring equal access to everywhere with an internet connection.
We have followed FOSS principles by making our code available on github in a public repository and the AI model we use is available on hugging face. AIDungeon and Character.ai are both closed source however.
